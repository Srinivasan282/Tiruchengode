# Tiruchengode – The Sacred Red Hill

A premium, modern, multi-page responsive tourism and city information website for Tiruchengode, Tamil Nadu.

**##DEMO Link:** https://srinivasan282.github.io/Tiruchengode/

## Features

- **8 Pages:** Home, About, Temples, Explore, Culture, Education, Travel, Contact
- **Design:** Minimal, premium UI with Tamil Nadu accent colors (saffron, red, gold)
- **Responsive:** Mobile, tablet, and desktop friendly
- **Animations:** Smooth scroll, hover effects, fade-in on scroll
- **SEO:** Meta tags, semantic HTML5

## Getting Started

1. Open `index.html` in a browser, or use a local server:


2. Add your images to `assets/images/` – see `assets/images/README.md` for the list of recommended images.

3. For production: Replace the map placeholder in `contact.html` with a Google Maps embed if desired.

## Customization

- **Colors:** Edit CSS variables in `assets/css/style.css` (`:root` section)
- **Content:** Update text directly in each HTML file
- **Footer:** The footer is included in each page; update as needed

## Browser Support

Modern browsers (Chrome, Firefox, Safari, Edge). Graceful degradation for older browsers.
